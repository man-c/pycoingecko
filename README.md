# CoinGecko API wrapper

Python wrapper around the [CoinGecko](https://www.coingecko.com/) API (V3)

### Installation
```
python3 setup.py install
```

### Test

Run unit tests with:

```
# after installing pytest using pip3
pytest tests
```

### Usage

```
from pycoingecko import CoinGeckoAPI
cg = CoinGeckoAPI()
```
